```html
<div altr-html="my_html"></div>
```
The `html` tag works exactly like the `text` tag, but sets the
[`innerHTML`](https://developer.mozilla.org/en-US/docs/Web/API/Element.innerHTML)
of the element instead of the `textContent`.
